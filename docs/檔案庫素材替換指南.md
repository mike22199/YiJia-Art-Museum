---

## 預覽

1. Live Server 開啟 `index.html`
2. 進入 `#archive/index` 或從首頁點「檔案庫」

## 編輯檔案

- **內容：** `content/site-content.json`
- **圖片：** `assets/images/…`

## 路由速查

| 頁面 | 網址 |
|------|------|
| 檔案庫首頁 | `#archive/index` |
| 歷屆展覽回顧 | `#archive/exhibitions?year=2026` |
| 藝術實踐 | `#archive/practice?year=2025` |
| 影音 | `#archive/media?year=2025` |
| 藝術家教師日誌 | `#archive/teachers?year=2026&teacher=t01` |
| 參考書目 | `#archive/bibliography` |

## JSON 區塊速查

| 頁面 | JSON 路徑 |
|------|-----------|
| 全站頁尾、子導覽 | `site`、`archiveSubnav` |
| 檔案庫首頁 | `archive.home` |
| 歷屆展覽 | `archive.exhibitionsByYear` |
| 藝術實踐 | `archive.practice.worksByYear` |
| 影音 | `archive.media.byYear` |
| 藝術家教師 | `archive.teachers` |
| 參考書目 | `archive.bibliography` |
