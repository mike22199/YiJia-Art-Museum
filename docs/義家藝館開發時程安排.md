# 義家藝館官方網站 — 開發時程安排

> 建立日期：2026-06-07  
> 用途：與業主／團隊對齊開發節奏、素材準備與驗收標準。  
> 技術基礎：現有靜態網站（HTML / CSS / JS + `content/site-content.json`，可選 Sanity CMS）

---

## 一、專案目標摘要

為「義家藝館」完成官方網站，包含：

1. **首頁**：進場動畫 → 主畫面互動（發光、Hover、三入口）
2. **特展01《推開自由門》**、**特展02《成為自由人》**：轉場動效 → 展覽理念 → 互動展間
3. **義家藝館檔案庫**：首頁 + 7 類子頁（歷屆展覽、藝術實踐、影音、典藏、研究、教師日誌、參考書目）

**開發策略**：分 Phase 逐步上線，每階段驗收後再進下一階段；內容與互動可「先上線一間房間／先上 2026 九位教師」再擴充。

---

## 二、現況與目標差距（2026-06-07）

| 區塊 | 目前已有 | 規格要求 | 主要差距 |
|------|----------|----------|----------|
| 首頁 | 可點擊熱區 + 下方導覽，部分 Hover | 進場動畫、發光、文字變紅、門 icon 打開 | 缺進場動畫；入口文案需對齊三展名 |
| 特展 | 靜態「關於展覽」+ 開幕展連結 | 側向放大、布幕動效、完整展間 | 缺轉場動畫與房間互動 |
| 檔案庫 | 時間軸式「歷年網站」 | 檔案庫首頁（Banner、精選影音等） | 需重做首頁版型 |
| 子導覽 | 歷年網站、藝術實踐、影音（3 項） | 7 項子頁 | 缺典藏、研究、教師日誌、參考書目 |
| 藝術家教師 | 研究頁翻書原型 | 2026 九位 + ISSUU 式日誌 | 需正式教師資料與日誌檔 |
| 素材 | 多為 placeholder | 各頁正式圖文影音 | 需依 Phase 分批提供 |

## 三、全站 Phase 總覽與建議時程

以下時程以「**每 Phase 約 1～2 週**」估算，實際長短取決於**素材是否齊全**與**回饋修正輪數**。

| Phase | 名稱 | 建議工期 | 前置條件（你方需提供） | 主要交付物 |
|-------|------|----------|------------------------|------------|
| **0** | 專案對齊 | 2～3 天 | 確認 PDF 架構、命名、優先順序 | 路由對照表、資料欄位說明 |
| **1** | 首頁動畫 + 互動 | 1～2 週 | 進場動畫、首頁主視覺、門 icon、三入口名稱 | 可上線首頁 |
| **2** | 特展01《推開自由門》 | 2～3 週 | 特展01 簡報、理念文案、第一間房間素材 | 轉場 + 理念頁 + 一間房 |
| **3** | 特展02《成為自由人》 | 2～3 週 | 特展02 簡報、同上結構素材 | 第二特展 + 兩展互切導覽 |
| **4** | 檔案庫首頁 + 歷屆展覽 | 2 週 | 首頁各區塊內容、歷屆作品試算表 | 檔案庫首頁 + 年份篩選作品網格 |
| **5** | 2026 教師 + 日誌 | 2～3 週 | 9 位教師資料、日誌 PDF／逐頁圖 | 教師頁 + ISSUU 式翻頁 |
| **6** | 藝術實踐、影音、參考書目 | 2 週 | 作品表、影音素材、10 本書資料 | 三個子頁可上線 |
| **7** | 擴充（後續） | 依內容 | 其他年份教師、典藏、研究素材 | 逐步補齊 |

### 累計時程參考（理想情況）

- **Phase 0～1 完成**（首頁可對外 Demo）：約 **2～3 週**
- **Phase 0～3 完成**（雙特展可體驗）：約 **7～10 週**
- **Phase 0～6 完成**（檔案庫主要功能齊備）：約 **12～16 週**
- **Phase 7**：依後續內容量另排

> 若某 Phase 素材延遲，可**跳過該 Phase 先做下一項**（例如特展素材未齊，可先做多 Phase 4 檔案庫靜態頁），但需先對齊以免路由或版型重工。

## 四、各 Phase 詳細說明

### Phase 0：專案對齊（2～3 天）

**目的**：避免後續改名、改架構造成重工。

**討論與確認項目**：

- [ ] 全站資訊架構是否與 PDF 一致
- [ ] 路由命名（例：`exhibition-left` → `push-door` 是否更名）
- [ ] 主導覽：共創／典藏／研究 是否改為「義家藝館檔案庫」單一入口
- [ ] 子導覽 7 項標籤與順序
- [ ] CMS 策略：繼續 JSON／Decap，或啟用 Sanity
- [ ] 素材資料夾結構與命名規則

**產出**：

- 路由與頁面對照表
- 各 Phase 素材清單（勾選版）
- 修正後的開發優先順序（若有調整）

---

### Phase 1：首頁動畫 + 首頁互動（1～2 週）

**規格對照（P5）**：

- 先播放進場動畫，結束後進入主畫面
- 紅框物件與下方文字可進入對應特展／檔案庫
- 可點物件有發光效果
- Hover：下方文字變紅；門 icon 變為打開狀態

**你方需提供**：

| 項目 | 格式 | 備註 |
|------|------|------|
| 進場動畫 | MP4 / GIF / Lottie 或關鍵幀 + 秒數 | 是否可跳過、是否只播一次 |
| 首頁主視覺 | PNG / JPG（建議 ≥1920px 寬） | 含三個可點區域 |
| 門 icon | SVG 或 PNG（關／開兩態） | Hover 用 |
| 三入口名稱 | 文字 | 推開自由門、成為自由人、義家藝館 |
| 發光效果參考 | 截圖或描述 | 顏色、強度、是否脈動 |

**驗收檢查**：

- [ ] 動畫播放正常，結束後進入主畫面
- [ ] 三個入口連結正確
- [ ] 發光與 Hover 符合設計
- [ ] 手機／平板基本可用（或確認支援範圍）

---

### Phase 2：特展01《推開自由門》（2～3 週）

**規格對照（P2、P6）**：

- 點選後頁面往旁放大、布幕放大顯示展覽理念
- 點「開幕展」進入完整展覽
- 導覽：回首頁、切換至「成為自由人」、分享社群
- **本階段範圍：先做一間房間**

**你方需提供**：

| 項目 | 說明 |
|------|------|
| 特展01 簡報 PDF | 展間配置、作品、互動說明 |
| 展覽理念文案 | 布幕頁文字 |
| 第一間房間 | 場景圖、可點物件座標、作品圖 + 說明 |
| 轉場參考 | 影片或文字描述節奏 |
| 社群分享 | 平台與連結 |

**驗收檢查**：

- [ ] 首頁 → 特展轉場動效
- [ ] 理念頁文案與「開幕展」進入展間
- [ ] 一間房間內互動與作品資訊正確
- [ ] 回首頁、切換特展02、分享按鈕可用

---

### Phase 3：特展02《成為自由人》（2～3 週）

結構同 Phase 2，另需**特展02 簡報**與專屬素材。

**額外驗收**：

- [ ] 特展01 ↔ 特展02 可互相切換
- [ ] 兩展視覺風格一致、內容不混淆

---

### Phase 4：檔案庫首頁 + 歷屆展覽回顧（2 週）

**規格對照（P4、8.1、8.2）**：

**檔案庫首頁區塊**：

- Header + 子 Header
- 主視覺 Banner 動畫
- 精選影音、特展入口、機構介紹
- 歷屆展覽回顧（含「了解更多＞」）
- 藝術家教師、教師作品
- Footer

**歷屆展覽回顧**：

- 年份：2016、2018、2019、2020、2021、2022、2026
- 作品網格：作品名、作品照、作者

**你方需提供**：

| 項目 | 格式建議 |
|------|----------|
| 檔案庫首頁各區文案與圖 | 分區標題 + 圖檔 |
| Banner 動畫 | 同 Phase 1 動畫格式 |
| 歷屆作品表 | 試算表：`年份｜作品名｜作者｜圖檔名｜簡介` |
| 機構介紹 | 一段正式介紹文 |

**驗收檢查**：

- [ ] 檔案庫首頁各區塊齊全
- [ ] 年份篩選與作品網格正確
- [ ] 子導覽可進入各子頁（尚未完成的可顯示「籌備中」）

---

### Phase 5：2026 年 9 位藝術家教師 + 日誌（2～3 週）

**規格對照（P8）**：

- 標題「2026年藝術家教師」
- 9 位：頭像 + 姓名（橫向排列）
- 教師詳情：姓名 + 簡介
- 日誌：ISSUU 式翻頁、可放大
- 年份 Tab：2019、2020、2021、2022、2026（其他年份可先空殼）

**你方需提供**：

| 項目 | 說明 |
|------|------|
| 9 位教師 | 姓名、頭像、簡介（每人） |
| 日誌檔 | 每位：PDF 或逐頁 JPG／PNG |
| ISSUU 參考連結 | 若有已上線版本 |

**驗收檢查**：

- [ ] 9 位教師卡片與詳情正確
- [ ] 翻頁動畫流暢、可放大檢視
- [ ] 年份 Tab 切換邏輯正確

---

### Phase 6：藝術實踐、影音、參考書目（2 週）

**規格對照（8.3、8.5、8.6、P9）**：

| 頁面 | 功能 |
|------|------|
| 藝術實踐 | 2025 年份、作者、關鍵字篩選；分類（網站／影音／海報／展覽概論）；教師頭像網格 |
| 影音 | 2025 年份、關鍵字搜尋、照片輪播、推薦文本 |
| 參考書目 | 10 本：封面 + 介紹文章，單頁展示 |

**你方需提供**：

- 藝術實踐作品與作者清單
- 影音照片、影片連結、推薦文本
- 10 本書：封面、書名、作者、介紹文（每本約 100～300 字）

**驗收檢查**：

- [ ] 篩選／搜尋功能正常
- [ ] 輪播與書目排版符合設計
- [ ] 10 本書內容完整

## 五、標準合作流程（每個 Phase 重複）

```
① 你提供該 Phase「素材包」
        ↓
② 開發實作（版型 + 互動 + 內容填入）
        ↓
③ 本地或 Netlify 預覽 → 你驗收（對照下方檢查清單）
        ↓
④ 你回饋（建議附截圖標註）
        ↓
⑤ 修正一輪 → Phase 結案 → 進入下一 Phase
```

**建議每 Phase 最多 2 輪免費修正**；若需求變更超出原規格，需重新對齊範圍與時程。

## 八、建議討論議題（下次會議）

請我們一起確認以下項目，再決定「從哪一 Phase 開工」：

1. **時程**：上述 12～16 週是否符合你方上線期望？是否有硬性截止日期（例如某展覽開幕日）？
2. **Phase 0**：路由與導覽命名是否要先開會定案？
3. **Phase 1 素材**：進場動畫、首頁圖、門 icon 目前準備狀態？
4. **並行策略**：特展素材與檔案庫素材哪邊先齊？是否調整 Phase 順序？
5. **CMS**：內容由誰長期維護？是否 Phase 4 起同步設定 Decap／Sanity？
6. **驗收方式**：本地預覽、Netlify 測試網址、或正式網域？
7. **範圍邊界**：手機版、無障礙、SEO 是否納入 Phase 1～6 或放 Phase 7？
